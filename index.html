Networking

Like Hardware, Networking is a core fundamental technology. Understanding networking principles makes it possible to interconnect hardware devices. Networking is the backbone of the internet as we know it today. With the advent of APIs (Application Programming interface) in networking and data transfer things keep getting more interesting. The ability to create user communication interfaces for programs written on top of chips and operating systems is just amazing.

Internet of Things

Getting to know hardware and networking fundamentals is just the beginning. IoT is where the fun lies, simply take that hardware knowledge and have fun. You do this by creating hardware objects and things that interconnect with each other. IoT raises the possibilites in our world.
